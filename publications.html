    Software development

    Guillerme T dispRity: a modular R package for measuring disparity. Methods in Ecology and Evolution. doi:10.1111/2041-210X.13022.

    v1.4:

    A modular package allowing to calculate disparity (as defined by user) matrices (typically from distance matrices from cladistic/morphometric/ecological data) and perform disparity analysis.

    Guillerme T & Weisbecker V. landvR: tools for measuring landmark position variation. ZENODO. 10.5281/zenodo.2620785.

    v0.5:

    This package contains tools for measuring, visualising and testing landmark variation.

    Guillerme T & Healy K. mulTree: a package for running MCMCglmm analysis on multiple trees. ZENODO. 10.5281/zenodo.12902.

    v1.3.7:

    Performs MCMCglmm on multiple phylogenetic trees. This package is based on the MCMCglmm package and runs a MCMCglmm analysis on multiple trees. Related publications: Healy et al. 2014, Proceedings B., Healy 2015, Proceedings B..

    Healy K., Guillerme T, Kelly S., Inger R., Bearhop, S., Jackson, A. SIDER: An R package for predicting trophic discrimination factors of consumers based on their ecology and phylogenetic relatedness. Ecography, doi:10.1111/ecog.03371

    v1.0.0.0:

    This package allows users to estimate Trophic Discrimination Factors (TDF) for species with no current measured TDF values using Bayesian imputation. This package is based on the MCMCglmm package and runs a MCMCglmm analysis on multiple trees using mulTree.

    Brazeau MD, Smith MR, Guillerme T. morphylib: a C library for calculating tree length under parsimony with inapplicable data. ZENODO. 10.5281/zenodo.815372.

    v1.0.0.0:

    This library allows to count tree length under our new inapplicable algorithm. This is part of the morphyproject.
